#### 模拟用户登陆

1. 先提示用户 是否 进行注册；
2. 然后进行接收用户登录信息；
3. 接收到信息，先判断用户名是否正确；
  - 当用户名正确，再判断密码是否正确
  - 密码不正确，提示用户或密码错误
  - 密码正确，提示登录成功
4. 用户名不正确，提示用户或密码错误
5. 判断用户输入账号密码的次数
  - 大于三次，则提示锁定，退出程序
  - 小于三次，可以继续尝试输入
